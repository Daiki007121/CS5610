5. Font Size Examples

  • 12px - Used for small print (e.g., footnotes)
  • 1em - Relative to parent size (e.g., normal text)
  • 1.5rem - Relative to root size (e.g., subheadings)
  • 150% - Percentage of parent (e.g., emphasis)
  • 16pt - Print-style sizing (e.g., documents)
  • 2vw - Viewport relative (e.g., responsive headlines)
